Etiquetas semanticas estructurales

div - Es una etiqueta contenedora que no tiene semantica
header - Cabecera de un sitio web o de una seccion
main- Define la seccion principal del documento, solo puede exisitir una etiqueta main por documento
nav- Representa una navegacion
article- Representa una seccion de autocontenido(que por si sola se explica)
aside- Respresenta contenido complementario o secundario
section- Representa una seccion de contenido genrico
address - representa una informacion de contacto
